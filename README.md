# 🏛️ Σύστημα Διαχείρισης Στοάς ΑΚΡΟΠΟΛΙΣ

Πλήρες σύστημα διαχείρισης μητρώου μελών για Μασονική Στοά.

## 📋 Περιεχόμενα

- **Μητρώο Μελών**: Διαχείριση 40 μελών με πλήρη στοιχεία
- **Επεξεργασία Μέλους**: Ενημέρωση προσωπικών, επικοινωνιακών, μασονικών στοιχείων
- **Καρτέλες PDF**: Δημιουργία επίσημων καρτελών μέλους (μεμονωμένες ή μαζικές)
- **Στατιστικά**: Αναλυτικά στοιχεία μητρώου

## 🚀 Τοπική Εκτέλεση

```bash
# Εγκατάσταση dependencies
pip install -r requirements.txt

# Εκτέλεση εφαρμογής
streamlit run lodge_management_app.py
```

## ☁️ Deployment στο Streamlit Cloud

### Βήμα 1: GitHub Repository

1. Δημιούργησε νέο repository στο GitHub
2. Upload τα αρχεία:
   - `lodge_management_app.py`
   - `requirements.txt`
   - `lodge_members.db`
   - `.gitignore`

### Βήμα 2: Streamlit Cloud

1. Πήγαινε στο [share.streamlit.io](https://share.streamlit.io)
2. Κάνε login με GitHub
3. Πάτα "New app"
4. Επέλεξε:
   - Repository: `your-username/your-repo`
   - Branch: `main`
   - Main file: `lodge_management_app.py`
5. Πάτα "Deploy"!

## 📊 Βάση Δεδομένων

Η βάση `lodge_members.db` περιέχει:

- **40 μέλη** της Στοάς ΑΚΡΟΠΟΛΙΣ
- **40+ πεδία** ανά μέλος:
  - Προσωπικά Στοιχεία (8 πεδία)
  - Στοιχεία Επικοινωνίας (6 πεδία)
  - Μασονικά Στοιχεία (11 πεδία)
  - Ιστορικό Στοάς (4 πεδία)
  - Οικογενειακά (5 πεδία)
  - Διοικητικά (6 πεδία)

## 📄 Καρτέλες PDF

Κάθε καρτέλα μέλους περιλαμβάνει:

- Πλήρη προσωπικά στοιχεία
- Μασονικό ιστορικό
- Οικογενειακά στοιχεία
- Διοικητική κατάσταση
- Υπογραφή Γραμματέα

Μορφοποίηση σύμφωνα με τον Κανονισμό Μεγάλης Στοάς της Ελλάδος.

## 🔐 Ασφάλεια

- Η βάση δεδομένων περιέχει ευαίσθητα προσωπικά δεδομένα
- Χρησιμοποιήστε private repository στο GitHub
- Μην μοιράζεστε το URL του deployed app δημόσια

Για ενημέρωση της βάσης δεδομένων:
